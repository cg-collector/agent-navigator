# 开源Agent项目学习 #055

## 项目名称
smolagents

## GitHub链接
https://github.com/huggingface/smolagents

## 一句话简介
Hugging Face 出品的极简 AI Agent 框架，核心逻辑仅约1000行代码，首创 CodeAgent 模式让 Agent 用代码思考。

## 项目架构

- **核心层**: `agents.py` 约1000行，最小化抽象，CodeAgent + ToolAgent 两种模式
- **CodeAgent**: Agent 直接用 Python 代码编写动作（而非自然语言），支持沙箱执行（Docker/E2B/Modal/WebAssembly）
- **模型层**: 模型无关，支持 HF Inference API、transformers 本地模型、Ollama、OpenAI、Anthropic 等（通过 LiteLLM）
- **工具层**: Tool-agnostic，支持 MCP Server 工具、LangChain 工具、HF Hub Space 作为工具
- **多模态**: 支持文本、视觉、视频、音频输入
- **Hub 集成**: Agent 和 Tool 可推送到 HF Hub 共享，支持 `push_to_hub()` / `from_hub()`

安装: `pip install "smolagents[toolkit]"`

## 技术亮点

1. **CodeAgent 范式**: Agent 用代码而非 JSON/文本思考，表达力更强、更精确、更易调试
2. **极简设计**: 1000行核心代码，开发者可以在几分钟内理解整个框架
3. **沙箱安全**: 支持 Docker、E2B、Modal、Pyodide+Deno WASM 多种沙箱环境
4. **HF Hub 生态**: Agent 即 Space，工具可共享，复用性极强
5. **MCP 原生支持**: 直接接入 MCP Server 工具，与 OpenClaw 的 MCP 生态兼容

## 对楠楠/学长的启发

- **Less is more**: 1000行代码做出一个完整的 Agent 框架，证明好的设计不需要复杂
- **Code > Text**: 用代码作为 Agent 的"思维语言"比自然语言更精确，这个思路值得关注
- **HF Hub 模式**: 工具和 Agent 作为可共享资源，类似 ClawHub 的思路但规模更大
- **MCP 生态融合**: smolagents 和 OpenClaw 都支持 MCP，未来工具生态可以互通
