# 开源Agent项目学习 #053

## 项目名称
Gemini CLI

## GitHub链接
https://github.com/google-gemini/gemini-cli

## 一句话简介
Google 官方开源的终端 AI Agent，把 Gemini 模型的能力直接带到命令行，支持 MCP 扩展、100万 token 上下文、内置工具链。

## 项目架构

Gemini CLI 采用 Terminal-first 设计，核心架构：

- **CLI 入口层**: 基于 Node.js，通过 `npx @google/gemini-cli` 即可运行，无需安装
- **模型层**: 直连 Gemini 3 系列，1M token 上下文窗口，支持多模态（文本、图片、PDF）
- **工具链层**: 内置 Google Search grounding（实时搜索）、文件操作、Shell 命令、Web 抓取
- **扩展层**: 完整 MCP（Model Context Protocol）支持，可接入 Imagen/Veo/Lyria 等媒体生成
- **配置层**: GEMINI.md 自定义上下文文件，类似 CLAUDE.md / AGENTS.md 的项目级配置
- **CI/CD 集成**: 提供 GitHub Action，自动 PR Review、Issue Triage

发布节奏：Nightly（每日）→ Preview（每周二）→ Stable（每周二）

## 技术亮点

1. **零门槛启动**: `npx` 直接运行，支持 npm/brew/conda 多种安装方式，5秒上手
2. **免费层慷慨**: 个人 Google 账号 60 req/min、1000 req/day，对开发者非常友好
3. **MCP 生态集成**: 不只是聊天工具，通过 MCP 可以扩展媒体生成、数据库查询等能力
4. **GitHub Action 自动化**: 把 AI Agent 嵌入 CI/CD 流程，自动做 Code Review 和 Issue 分类
5. **会话检查点**: 支持保存和恢复复杂会话，适合长时间调试任务
6. **1M token 上下文**: 可以直接分析整个大型代码库

## 对楠楠/学长的启发

- **GEMINI.md 设计思路值得借鉴**: 和 OpenClaw 的 AGENTS.md / SOUL.md 思路一致，项目级配置文件是 AI Agent 的标配
- **GitHub Action 集成**: 可以考虑给 OpenClaw 写类似的 GitHub Action，自动 review PR
- **MCP 生态是趋势**: Google、OpenAI 都在支持 MCP，OpenClaw 的 MCP 支持方向是对的
- **免费层策略**: Google 用免费额度吸引开发者生态，这种 go-to-market 策略值得注意
