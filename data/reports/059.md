# 开源Agent项目学习 #059

## 项目名称：Mastra

## GitHub 链接
https://github.com/mastra-ai/mastra

## 一句话简介
由 Gatsby 团队打造的 TypeScript AI Agent 框架，提供 Agent、Tools、Workflows、Memory 等完整原语，主打 TypeScript 生态的原生体验。

## 核心架构
- **技术栈**: TypeScript / Node.js
- **设计模式**:
  - Agent = LLM + Tools + Memory，支持内部推理循环
  - Workflow：可视化步骤编排，支持 Agent 和 Tool 的组合调用
  - Memory：内置对话记忆和 RAG 支持
  - 模块化原语：Agent、Tool、Workflow、Memory 可独立使用也可组合

## 技术亮点
1. **TypeScript 原生**：对 TS 开发者来说类型安全、IDE 补全体验极佳，不像 Python 框架需要切换语言
2. **Gatsby 团队出品**：团队有构建开发者工具的经验，API 设计成熟
3. **Workflow 编排**：支持可视化步骤组合，Agent 可以作为 Workflow 的一个步骤嵌入
4. **内置 RAG**：Memory 模块自带向量检索，开箱即用
5. **全栈友好**：前后端统一 TS，适合 Next.js 等全栈项目集成

## 对楠楠/学长的启发
- TypeScript Agent 框架是个好方向——学长如果做前端项目，可以用 Mastra 而不用切 Python
- Workflow + Agent 的组合编排思路比纯 Agent 链更灵活
- OpenClaw 本身是 Node.js 生态，Mastra 的设计模式可以参考

## 评分：⭐⭐⭐⭐
TypeScript 生态的强力选择，但目前社区规模还小于 LangChain/CrewAI，需要观察发展。
