## 简介

OmAgent 是一个用于构建**多模态语言智能体（Multimodal Language Agents）**的 Python 库，由OmAI Lab 开发，发表在 EMNLP 2024。

它的核心理念是：**让构建多模态 AI Agent 变得简单**——不追求大而全的框架抽象，而是把复杂的工程细节（Worker 编排、任务队列、节点优化等）隐藏在幕后，提供一个极易上手的接口来定义 Agent。

OmAgent 最初的应用场景是**视频理解**——构建能回答任意关于上传视频的问题的智能体。后来扩展为通用的多模态 Agent 开发框架。

---

## 架构设计

### 1. Agent Operators（智能体算子）

OmAgent 最具创新性的设计是**Agent Operators**——将各种推理策略（CoT、ReAct、SC-CoT 等）封装为可复用、可比较的算子。

这相当于把 Agent 的"思考模式"变成了可插拔的模块，方便对比和组合。

### 2. 图化工作流编排引擎

OmAgent 提供**基于图的工作流编排引擎**，用 `container.yaml` 配置文件来组织 Agent 的组件和依赖关系，支持复杂的多步骤工作流。

### 3. 多模态原生支持

- 原生支持 VLM（视觉语言模型）推理
- 支持视频处理
- 支持移动设备连接
- 支持实时 API 调用
- 支持本地模型部署（Ollama、LocalAI）

### 4. 分布式架构

- 支持完全分布式部署
- 支持横向扩展
- 提供 **Lite 模式**，无需中间件即可运行

## 技术亮点

### 1. SC-CoT（Self-Consistent Chain of Thought）

在 Open Agent Leaderboard 上，SC-CoT 在 GSM8K 和 AQuA 测试集上表现最优，超越了 CoT 和 ReAct。这是一种带自洽性的思维链方法。

性能对比（GPT-3.5-turbo）：
| 算法 | GSM8K 得分 | AQuA 得分 |
|------|-----------|-----------|
| SC-CoT | 73.69 | 67.32 |
| CoT | 69.86 | 61.02 |
| ReAct-Pro | 69.74 | 64.57 |
| IO（基线） | 38.40 | 38.98 |

### 2. 多模态工作流

OmAgent 的视频理解 Agent 示例非常亮眼——用户上传一段视频，可以问任何关于视频内容的问题，背后是"任务分解"（Divide-and-Conquer）的设计思路。

### 3. 低门槛快速原型

```bash
pip install omagent-core
cd examples/step1_simpleVQA
python compile_container.py
python run_webpage.py
# 打开 http://127.0.0.1:7860 即可体验
```

三步就能跑起来一个多模态 VQA 应用，门槛极低。

### 4. Agent Operators 排行榜

OmAgent 团队维护了一个 [open-agent-leaderboard](https://github.com/om-ai-lab/open-agent-leaderboard)，在统一的测试环境和模型下对比各种推理策略的效果和成本，帮助研究者做出选择。

## 对楠楠/学长的启发

### 1. **多模态是未来，楠楠要重视**

OmAgent 证明了多模态 Agent（文本+图片+视频+音频）的实用价值。学长做开发的话，多模态 Agent 会是下一个重要方向。

### 2. **推理策略的可比较性很重要**

Agent Operators + Leaderboard 的思路很值得学习——把不同的推理策略（CoT、ReAct、SC-CoT）标准化后对比，不仅方便研究，也能帮助实际选型。

### 3. **Lite 模式降低部署门槛**

OmAgent 提供无需中间件的 Lite 模式，这个思路很务实。学长如果做工具，能简化部署就简化，别让用户为了跑一个小工具还要搭一套 infrastructure。

### 4. **视频理解 Agent 是一个好方向**

随着视频内容爆炸，能理解视频、回答视频相关问题的 Agent 会越来越有价值。这个方向值得楠楠关注。
