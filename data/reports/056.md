# 开源Agent项目学习 #056

## 项目名称
Crawl4AI

## GitHub链接
https://github.com/unclecode/crawl4ai

## 一句话简介
GitHub #1 趋势开源爬虫项目，专为 LLM/Agent 设计的 AI-Ready 网页抓取框架，自动将网页转为结构化 Markdown。

## 项目架构

- **爬取引擎**: 高性能异步爬虫，支持 JavaScript 渲染
- **内容提取层**:
  - `DefaultMarkdownGenerator`: 基础 HTML → Markdown 转换
  - `fit_markdown`: 智能内容提取，自动识别正文、去除噪音（广告、导航等）
  - `LLMExtractionStrategy`: 利用大模型从网页提取结构化信息
- **输出层**: 支持 Markdown、Cleaned HTML、结构化 JSON 等多种格式
- **集成层**: 可作为 Agent 工具使用，支持向量搜索索引

## 技术亮点

1. **AI-First 设计**: 不是通用爬虫，而是专门为 LLM 训练数据和 Agent 输入优化
2. **fit_markdown 智能提取**: 自动过滤噪音内容，提取核心正文，质量接近商业方案
3. **LLMExtractionStrategy**: 用大模型做结构化信息提取，比正则/CSS选择器灵活得多
4. **GitHub #1 Trending**: 社区活跃度高，持续迭代
5. **向量搜索集成**: 爬取内容可直接用于 RAG 系统的索引构建

## 对楠楠/学长的启发

- **Agent 需要好的数据管道**: Crawl4AI 解决了 Agent 获取网页信息的前置问题
- **结构化 > 原始HTML**: 给 LLM 喂 Markdown 比喂 HTML 效果好很多，OpenClaw 的 web_fetch 也用了类似思路
- **LLM + 传统方法结合**: LLMExtractionStrategy 是混合方案的典范
- **可以集成到 OpenClaw**: 作为 MCP 工具或 Skill 引入，增强 Agent 的信息获取能力
