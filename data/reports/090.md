## 项目简介

Semantic Kernel 是微软开源的轻量级 AI Agent 开发工具包（SDK），让开发者能够轻松地将大语言模型（LLM）集成到现有应用中，并构建单 Agent 或多 Agent 系统。

它的核心定位是**企业级、可扩展、模型无关**的 Agent 框架，既适合快速原型也适合生产部署。

---

## 架构设计

### 核心理念：Kernel（内核）

Semantic Kernel 的核心是 **Kernel（内核）**，它充当 LLM 与应用代码之间的"粘合层"：

```
用户代码 → Kernel → LLM（可插拔：OpenAI / Azure OpenAI / HuggingFace / Ollama 等）
         ↓
       Plugins（工具/技能）
         ↓
       Memory（向量存储）
         ↓
       Planner（规划能力）
```

### 关键组件

1. **Kernel（内核）**：核心编排引擎，管理所有组件的注册与调用
2. **Plugins（插件）**：封装工具函数，支持原生代码、Prompt 模板、OpenAPI 规范、以及 MCP（Model Context Protocol）
3. **Memory（记忆）**：支持向量数据库（Azure AI Search、Elasticsearch、Chroma 等），实现 RAG
4. **Planner（规划器）**：让 Agent 自动分解任务并调用工具
5. **Agent Framework**：构建模块化 Agent，支持 ChatCompletionAgent 等多种类型
6. **Process Framework**：用结构化方式建模复杂业务流程

### 多 Agent 编排

支持多个专业 Agent 协同工作，形成多 Agent 系统，适合复杂企业场景。

## 技术亮点

### 1. 模型无关（Model-Agnostic）
一套代码可以无缝切换底层 LLM：OpenAI、Azure OpenAI、Hugging Face、NVIDIA NIM、Ollama（本地部署）、LMStudio、ONNX 等。

### 2. 多语言支持
同时支持 **Python、.NET（C#）、Java** 三大生态，企业可以根据现有技术栈选择。

### 3. Plugin 生态 + MCP 支持
- 原生代码函数（Python Decorator / C# Attribute）
- Prompt 模板
- OpenAPI 规范导入
- **MCP（Model Context Protocol）** 一流支持，可连接各种外部数据源和工具

### 4. 企业级特性
- 完整的可观测性（Telemetry）
- 安全机制
- 稳定 API（相对 LangChain 更稳定）
- 结构化输出（基于 Pydantic / C# class）

### 5. 本地部署友好
支持 Ollama、LMStudio、ONNX，可在离线/私有化环境中运行，不依赖云端 API。

### 6. 与微软生态深度集成
与 Azure AI Search、Azure OpenAI、Teams 等微软产品天然集成，适合企业场景。

## 对楠楠/学长的启发

### 对学长（技术视角）：

1. **Plugin 机制值得借鉴**：SK 的 `@kernel_function` 装饰器模式非常优雅，可以参考这种思路设计楠楠的"技能/工具"注册机制，让不同工具像插件一样可插拔。

2. **多语言 SDK 的价值**：SK 同时维护 Python / .NET / Java 三套 SDK，体现了"一次设计、多语言实现"的工程价值。楠楠如果要做自己的 Agent 框架，也可以考虑这种跨语言架构思维。

3. **企业级稳定性**：SK 相比 LangChain 更"opinionated"（有主见），API 稳定性和可测试性更好。学长在做自己的 Agent 系统时，稳定性可能比灵活性更重要。

4. **MCP 是未来**：SK 对 MCP 的一流支持说明 Model Context Protocol 正成为 Agent 工具调用的标准协议。

### 对楠楠（产品/学习视角）：

1. **工具即技能**：把每个工具想象成楠楠的一个"技能"（会做饭、会查天气、会撒娇……），用统一的框架管理，会让 Agent 更有组织。

2. **规划能力**：SK 的 Planner 概念让 Agent 自己决定下一步做什么，楠楠也可以思考——如果有一个"规划大脑"来安排楠楠每天的学习和生活任务，会很酷！

3. **本地部署**：SK 支持 Ollama 本地运行，意味着可以不依赖外部 API。学长给楠楠部署的 OpenClaw 其实也在朝这个方向走。

---

## 总结

Semantic Kernel 是目前最成熟的企业级开源 AI Agent 框架之一，由 Microsoft 背书，支持多语言、插件化、MCP、稳定 API，非常适合需要生产级可靠性的项目。相比 LangChain 的"灵活性优先"，SK 更注重"稳定性优先"，各有适用场景。

学长在做 Agent 开发时，Semantic Kernel 的架构理念（Kernel + Plugins + Memory + Planner）是非常值得参考的设计范式。
