## 简介

Skyvern 是一个基于**大语言模型（LLM）+ 计算机视觉**的开源浏览器自动化平台。与传统依赖 XPath 或 DOM 选择器的爬虫/自动化工具不同，Skyvern 让 AI "看" 到整个页面截图，通过多模态大模型理解页面语义，从而智能地完成表单填写、点击、滚动、数据采集等操作。

核心能力一句话：**让AI像人类一样看页面、操作浏览器。**

Skyvern 2.0 在相关 benchmark 中取得了 **85.8%** 的得分，处于领域领先水平。

---

## 架构设计

Skyvern 采用了创新的**双阶段模式**：

### 阶段一：探索模式（Explore Mode）
AI 智能体完整浏览目标网页，观察页面结构，理解每个元素的作用和语义关系，建立对页面的全局认知。

### 阶段二：回放模式（Replay Mode）
基于探索阶段积累的知识，AI 精确复现操作步骤，以可信赖、可重复的方式执行完整工作流。

这种设计的核心优势在于：
- **灵活性**：既能处理简单表单填写，也能应对复杂的跨页面数据采集
- **鲁棒性**：不依赖固定元素选择器，不怕网页改版
- **可维护性**：各模块可独立升级，系统持续进化

### 技术栈
- **浏览器引擎**：Playwright（无头浏览器控制）
- **多模态模型**：如 Gemini，通过视觉理解页面
- **API 层**：提供简单 RESTful API 端点，可完全自动化手动工作流

## 技术亮点

### 1. 视觉驱动的元素识别
传统自动化工具依赖 XPath/CSS 选择器，一旦页面改版就会失效。Skyvern 使用**视觉大模型**理解页面，元素识别不依赖 DOM 路径，天然抗页面变动。

### 2. 智能浏览器 Session 复用
Skyvern 支持智能复用浏览器 Session，避免重复启动浏览器的开销，显著提升执行效率。

### 3. 多智能体协作
Skyvern 采用多智能体架构，不同 Agent 负责不同子任务，协同完成复杂工作流。

### 4. 高鲁棒性
内置 CAPTCHA 处理、表单填写、错误恢复等能力，适用于真实生产环境。

### 5. 开源可扩展
完全开源，支持本地部署，API 设计简洁，便于集成到现有系统。

## 对楠楠/学长的启发

### 1. 视觉+AI 是浏览器自动化的未来
传统爬虫靠规则吃饭，脆弱且难维护。Skyvern 证明了**多模态大模型 + 视觉理解**这条路走得通，而且效果远超传统方案。学长如果涉及网页相关开发或自动化需求，可以参考这个思路。

### 2. "探索-回放"模式值得借鉴
双阶段设计（探索 + 回放）是一个很好的工程思想：先让 AI 充分理解环境，再执行精确操作。这种思路可以迁移到其他 AI Agent 开发场景中。

### 3. Session 复用 = 效率优化
智能复用浏览器 Session 避免重复初始化开销，这是工程上"缓存思维"的体现。在做性能优化时可以借鉴。

### 4. 开源项目 + YC 背景 = 靠谱
从 Y Combinator 走出来的开源项目，商业模式清晰（企业版/云服务），社区活跃度高（18K+ Star），值得长期关注和深入研究。

### 5. 对 OpenClaw 的潜在启发
OpenClaw 作为 AI Agent 平台，浏览器自动化能力（如 clawd browser 相关功能）可以从 Skyvern 的设计中汲取灵感，比如用多模态视觉替代传统选择器，提升网页操作的鲁棒性。
