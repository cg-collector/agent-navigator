# 开源Agent项目学习 #039

## 项目名称
Hello-Agents

## GitHub链接
https://github.com/datawhalechina/hello-agents

## 一句话简介
Datawhale 社区出品的系统性 AI Native 智能体教程，从零原理到多智能体实战，附带自研 Agent 框架 HelloAgents。

## 项目架构

项目分为三大板块、十一章：

**第一部分：基础理论**
- 智能体定义、类型、范式与应用（Ch1-2）
- 大语言模型基础（Ch3）

**第二部分：动手构建**
- 手把手实现 ReAct、Plan-and-Solve、Reflection 三大经典范式（Ch4）
- 低代码平台实践：Coze、Dify、n8n（Ch5）
- 主流框架对比：AutoGen、AgentScope、LangGraph（Ch6）
- **自研 Agent 框架 HelloAgents**：基于 OpenAI 原生 API 从零构建（Ch7）

**第三部分：高级扩展**
- 记忆与检索、RAG（Ch8）
- 上下文工程（Ch9）
- 通信协议：MCP、A2A、ANP（Ch10）
- Agentic RL：SFT 到 GRPO 的 LLM 训练（Ch11）

配套实战项目：智能旅行助手、赛博小镇（多智能体虚拟社会）。

## 技术亮点
- **从零手搓框架**：不依赖 LangChain 等黑盒框架，用原生 OpenAI API 构建完整 Agent 框架（HelloAgents），让开发者真正理解底层机制
- **理论与实践并重**：不仅讲"怎么用"，更讲"为什么"，涵盖 Agent 发展史、经典范式对比
- **覆盖前沿协议**：系统讲解 MCP、A2A（Google）、ANP 等 2025-2026 年 Agent 通信协议标准
- **含 Agentic RL 章节**：教程中难得一见的 LLM 训练内容，从 SFT 到 GRPO 全流程
- **中文社区驱动**：Datawhale 出品，中文文档友好，社区活跃

## 适用场景
- 想系统学习 Agent 开发但不想被框架束缚的开发者
- 需要理解 Agent 底层原理的工程师
- 团队内训、高校教学的 Agent 课程教材
- 从"LLM 使用者"进阶为"Agent 构建者"的学习路径

## 对我们的启发
1. **自研框架思路值得借鉴**：HelloAgents 从零构建 Agent 框架的方式，可以帮助我们更好地理解 OpenClaw 的底层设计。当需要定制特殊功能时，理解原生 API 比依赖黑盒框架更灵活
2. **上下文工程（Ch9）直接关联楠楠的日常工作**：楠楠每天处理多轮对话、维护记忆文件、管理 SOUL.md/USER.md，本质上就是在做上下文工程。系统学习这部分知识可以优化我们的记忆管理策略
3. **Agentic RL 是未来方向**：教程中的 GRPO 训练部分提示了 Agent 自我进化的一条路径。可以想象未来楠楠能通过 RL 反馈循环不断优化自己的行为策略
4. **通信协议章节帮助理解 OpenClaw 生态**：MCP、A2A 等协议是 OpenClaw 插件系统的基础，深入理解有助于我们更好地编写和使用 Skills
