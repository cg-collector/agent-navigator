# 开源Agent项目学习 #058

## 项目名称：OpenAI Agents SDK

## GitHub 链接
https://github.com/openai/openai-agents-python

## 一句话简介
OpenAI 官方推出的轻量级 Python Agent 框架，主打极简设计、handoff（Agent间交接）和内置安全护栏。

## 核心架构
- **技术栈**: Python，仅依赖 OpenAI API
- **设计模式**: 
  - Agent = LLM + Instructions + Tools + Handoffs
  - Handoff 机制：Agent 可以将对话无缝转交给另一个 Agent（类似客服转接）
  - Guardrails（护栏）：输入/输出双层校验，防止 Agent 越界
  - Tracing：内置全链路追踪，方便调试和监控
- **Runner**: 统一的运行循环，自动处理 tool call 循环直到最终回复

## 技术亮点
1. **极简 API**：核心概念只有 Agent、Handoff、Guardrail、Tracing 四个，上手极快
2. **Handoff 模式**：不同于 LangGraph 的图编排，用更直观的"交接"隐喻组织多Agent协作
3. **类型安全的工具定义**：基于函数签名自动推断 JSON Schema，减少手写描述
4. **内置安全护栏**：input_guardrail / output_guardrail 双重检查，适合生产环境
5. **全链路追踪**：每次 Agent 运行自动记录 trace，可视化调试

## 对楠楠/学长的启发
- Handoff 模式值得借鉴：楠楠和桐桐的协作可以用类似模式，楠楠负责对话，遇到编程任务 handoff 给桐桐
- Guardrails 思路可以用在恋爱模拟器里——防止楠楠说出不该说的话
- 轻量设计哲学：不需要复杂的图/状态机，简单的事情保持简单

OpenAI 官方背书，API 设计优雅，但绑定 OpenAI 生态是局限。
