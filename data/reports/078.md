**GitHub**: https://github.com/google-gemini/gemini-cli  
**Stars**: 96.7K+（全球 Top 100 GitHub 项目）  
**License**: Apache 2.0

---

## 一句话简介

Gemini CLI 是 Google 官方发布的开源 AI Agent，直接在终端里调用 Gemini 大模型，让开发者用自然语言完成代码编辑、调试、文件操作、Git 工作流等任务。支持 MCP（Model Context Protocol）扩展，1M token 超长上下文免费用。

---

## 架构设计

Gemini CLI 采用**命令式终端交互 + 模块化工具层**架构：

```
┌─────────────────────────────────────────────┐
│            Terminal UI (Readline)            │
├─────────────────────────────────────────────┤
│          Command Parser & Dispatcher         │
│  (/help, /chat, /code, /search...)           │
├─────────────────────────────────────────────┤
│         Agent Core (Gemini 3 Model)          │
│  - 1M token context window                   │
│  - 多轮对话状态管理                           │
│  - Conversation checkpointing（会话存档/恢复） │
├─────────────────────────────────────────────┤
│           Tool System (Built-in)              │
│  - File operations（读/写/搜索文件）           │
│  - Shell command execution（执行终端命令）     │
│  - Web fetching（网页抓取）                    │
│  - Google Search grounding（实时搜索）         │
│  - Git operations（PR查询、rebase等）           │
├─────────────────────────────────────────────┤
│        MCP Layer (Extensibility)              │
│  - Model Context Protocol 支持               │
│  - 社区扩展：Imagen/Veo/Lyria 媒体生成等       │
└─────────────────────────────────────────────┘
```

### 核心交互模式

1. **交互式对话**：进入项目目录 → `gemini` → 自然语言指令
2. **静默脚本**：配合 `--output-format json/stream-json` 做 CI/CD 集成
3. **上下文文件 GEMINI.md**：项目根目录放配置文件定制 Agent 行为

## 技术亮点

### 1. 超大上下文 + 免费层
- **1M token** 上下文窗口（Gemini 3），可分析整个代码库
- 免费套餐：**60请求/分钟，1000请求/天**（Google 账号）
- 支持直接读 PDF、图片、草图等多模态输入

### 2. 深度 Git 集成
- 查询 PR 变更、自动化 code review、issue triage
- GitHub Action 原生集成（`@gemini-cli` 可在 PR/Issue 里 @ 提问题）
- 复杂 rebase 操作支持

### 3. MCP（Model Context Protocol）扩展
- Google 官方推进的开放协议
- 社区可编写 MCP Server 扩展工具能力
- 已有的扩展：Vertex AI Creative Studio（Imagen/ Veo 媒体生成）

### 4. 多认证方式
| 方式 | 适用场景 | 特点 |
|------|----------|------|
| Google 账号登录 | 个人开发者 | 免费额度最大，无 API Key 管理 |
| `GEMINI_API_KEY` | 开发者（需自备Key） | 可选模型，1000次/天免费 |
| Google Cloud | 企业团队 | 高限额、合规、GCP 集成 |

### 5. 对话存档与恢复
- `conversation checkpointing`：长会话可保存/恢复，不丢上下文
- 配合 `GEMINI.md` 定制项目级行为

## 对楠楠/学长的启发

### 1. 小而美的工具哲学值得借鉴
Gemini CLI 核心逻辑约 1000 行（smolagents 也是类似思路），没有过度设计。对比一些臃肿的 Agent 框架，**够用就行**的极简主义反而让代码更可靠、更易调试。

### 2. 多认证分层是商业化标配
Gemini CLI 做了三层认证：免费 → 开发者付费 → 企业。这种分层策略让同一产品覆盖从个人到企业的所有用户群体，是一个值得参考的产品设计思路。

### 3. 终端是开发者的「第一界面」
Gemini CLI 选择终端而不是 Web UI，说明**CLI 工具在开发者生态里有不可替代的地位**。楠楠如果要做开发辅助工具，优先做好命令行接口会让工具更有生命力。

### 4. MCP 协议可能成为 Agent 扩展的事实标准
Model Context Protocol 正在获得 Google、OpenAI 等多方支持。以 MCP 为核心扩展生态，而非自己发明轮子，是一个聪明的生态策略。

### 5. 超大上下文 + 免费 = 杀手锏
1M token 免费额度意味着用户可以把整个代码库丢给 Agent 而不需要 RAG。这是让 AI 编程助手从「辅助」变成「主力」的关键一步。

## 总结

Gemini CLI 是一个**终端优先、社区驱动、超大上下文免费**的开源 AI Agent 项目。96K+ stars 的增长势头说明它精准命中了开发者"想要直接在终端里用 Gemini"的刚需。其设计理念——极简核心 + 可扩展 MCP 协议 + 多层认证——是一个值得学习的商业开源项目范本。
